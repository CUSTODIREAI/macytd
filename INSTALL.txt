YouTube ULTIMATE Downloader 2025 - macOS Installation
================================================================================

SYSTEM REQUIREMENTS:
- macOS 10.14 (Mojave) or later
- Python 3.7 or later (Python 3.7-3.9 recommended for Mojave)
- 100MB free disk space
- Intel or Apple Silicon compatible

================================================================================
INSTALLATION - CHOOSE YOUR macOS VERSION
================================================================================

FOR MOJAVE (10.14) / CATALINA (10.15) - INTEL MACS:
---------------------------------------------------
Use the Mojave-specific installer:

1. Open Terminal
2. Navigate to this folder:
   cd /path/to/downloaded/folder
3. Run the Mojave installer:
   bash install-mojave.sh

The script will automatically detect your system and install compatible versions.

FOR BIG SUR (11.0) AND LATER - APPLE SILICON & INTEL:
------------------------------------------------------

1. COPY FILES TO YOUR MAC
   - Drag "YouTube-Downloader.py" to your Desktop or Applications folder
   - Drag "launch-youtube-downloader.sh" to the same location

2. FIRST TIME LAUNCH
   - Double-click "launch-youtube-downloader.sh"
   - OR open Terminal and run:
     bash launch-youtube-downloader.sh

   The launcher will automatically:
   - Check for Python 3
   - Install required dependencies (PyQt5, yt-dlp)
   - Launch the YouTube Downloader

3. SUBSEQUENT LAUNCHES
   - Just double-click "launch-youtube-downloader.sh"
   - Or run: python3 YouTube-Downloader.py

================================================================================
MANUAL INSTALLATION (Advanced Users)
================================================================================

If you prefer to install dependencies manually:

1. Open Terminal

2. Install Python 3 (if not already installed):
   brew install python3

3. Install dependencies:
   pip3 install PyQt5 yt-dlp

4. Run the application:
   python3 YouTube-Downloader.py

================================================================================
TROUBLESHOOTING
================================================================================

Issue: "Python 3 Required" alert
Fix: Install Python 3:
     /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
     brew install python3

Issue: Dependencies won't install
Fix: Try manual installation:
     pip3 install --user PyQt5 yt-dlp

Issue: "Permission denied"
Fix: Make scripts executable:
     chmod +x launch-youtube-downloader.sh
     chmod +x YouTube-Downloader.py

Issue: Downloads fail with 403 errors
Fix: Click "Check & Update yt-dlp" in the app

Issue: App won't launch
Fix: Run from Terminal to see errors:
     python3 YouTube-Downloader.py
